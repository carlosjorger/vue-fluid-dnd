Fluid DnD is a [**fluid**, **agnostic** and **versatil** drag and drop
library for lists with support for Vue 3 and Svelte](https://fluid-dnd.netlify.app/). It's a**lightweight** tool ~7 Kb (gzip) with no depenencies.
